# People Intelligence - PropTech Connect

AI-powered research tool for understanding anyone before your next conversation.

## Features

- Input person details (name, LinkedIn, company info)
- AI generates tailored intelligence reports
- Context-aware (Meeting, Sales, Partnership, Marketing)
- Quick Brief or Full Deep Dive modes
- Save people to browser localStorage
- Export reports as JSON

## Setup

```bash
npm install
npm start
```

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Deploy (auto-detects Create React App)
